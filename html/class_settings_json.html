◆ toString()

std::string SettingsJson::toString ( uint32_t  opt = JsonOpt::NO_OPT,
uint32_t  tabOffset = 0,
bool  termWithComma = false 
) const

Convert SettingsJson to string.

You can change print options:

  • JsonOpt::NO_OPT -> default value (just print the json)
  • JsonOpt::DISCARD_DISABLED -> dont print values disard in file (used to print the variable in a file)
  • JsonOpt::VERBOSE -> verbose mode: print description, min, max, ...
  • JsonOpt::COLOR -> enable color to print
Parameters
optAll the option, you can use pipe to set multiple options (JsonOpt::VERBOSE | JsonOpt::COLOR)
tabOffsetDon't use this option
termWithCommaDon't use this option
Returns
std::string The SettingsJson object as string
