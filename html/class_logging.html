Detailed Description

manage logging in all project

To use Logging, you need to call the defines:

  • logDebug
  • logInfo
  • logSuccess
  • logWarn
  • logErr
  • logFatal
  • logErrExit
  • logFatalExit For example `logDebug("fps: " << fps);
