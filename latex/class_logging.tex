\doxysubsection{Detailed Description}
manage logging in all project 

To use \mbox{\hyperlink{class_logging}{Logging}}, you need to call the defines\+:
\begin{DoxyItemize}
\item log\+Debug
\item log\+Info
\item log\+Success
\item log\+Warn
\item log\+Err
\item log\+Fatal
\item log\+Err\+Exit
\item log\+Fatal\+Exit For example \`{}log\+Debug(\char`\"{}fps\+: \char`\"{} $<$$<$ fps); 
\end{DoxyItemize}
